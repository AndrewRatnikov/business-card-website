# Business Card Website

This is a business card website for a strategist for experts, built with Astro.js and Tailwind CSS.

## 🚀 Project Setup

1.  **Install dependencies:**

    ```bash
    npm install
    ```

2.  **Run the development server:**

    ```bash
    npm run dev
    ```

    This will start a local development server at `http://localhost:4321`.

## ⚙️ Building for Production

To build the site for production, run the following command:

```bash
npm run build
```

This will create a `dist/` directory with the optimized, static files for your site.

## 🧐 Preview

To preview the production build locally, run the following command:

```bash
npm run preview
```

This will start a local server to preview the contents of the `dist/` directory.

## 🔥 Deploying to Firebase Hosting

This site is configured to be deployed to Firebase Hosting.
